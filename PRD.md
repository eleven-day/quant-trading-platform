# 产品需求文档（PRD）

## 一、项目概述
量化交易学习平台是一款全栈应用，面向量化交易爱好者与学习者，提供从数据获取、策略开发、回测分析到可视化展示与学习资源的完整闭环环境。后端以 **FastAPI** + **Akshare** 为核心，前端采用 **React**（基于 Create React App），整体设计遵循原子化、模块化原则，保证可调试、易复用、易迭代。

## 二、目标与范围
- 提供一站式量化交易学习与实战平台。
- 数据获取：基于 Akshare 支持股票、基金、期货、外汇等多品种历史与实时数据。
- 策略开发：内置代码编辑器、示例策略、语法高亮、版本控制。
- 回测分析：多维度性能指标、可视化报表与图表。
- 可视化仪表盘：交互式图表，支持定制化布局。
- 学习资源：文档、示例数据、示例策略、视频教程链接。

**不在本版本范围**：实盘交易接口、复杂订单管理、高频微秒级策略。

## 三、用户角色与使用场景
1. 学习者：阅读资源、运行示例策略、查看回测结果与可视化图表。
2. 开发者/策略研究者：在平台内编辑、调试、管理自定义策略、对比不同版本表现。
3. 数据分析师：浏览与导出历史数据、生成自定义图表。

## 四、功能需求
### 4.1 后端模块
#### 4.1.1 数据服务（Data Service）
- 原子化接口：`/api/data/{asset}/{symbol}/history`、`/api/data/{asset}/{symbol}/realtime`
- 参数化查询：支持周期（daily, weekly）、起止日期、数据字段过滤
- 缓存与限流：简单内存/Redis 缓存、并发数限制

#### 4.1.2 策略开发与回测（Strategy & Backtest）
- 策略模板管理：增删改查、自带示例样板
- 策略运行：沙箱环境中加载、测试执行；异步任务队列（Celery/内置线程池）
- 回测框架：按日期区间回测，返回绩效指标（净值、夏普率、最大回撤等）
- 回测结果存储与检索API：`/api/backtest/{id}/result`

#### 4.1.3 学习资源管理（Docs Service）
- 文档目录接口：Markdown 存储、树状索引、全文检索
- 资源类型：文章、示例代码、视频链接

#### 4.1.4 用户与权限（Auth Service）
- 用户注册/登录（JWT）
- 基础权限：游客、注册用户、管理员
- 接口鉴权、中间件原子化实现

#### 4.1.5 系统监控与日志
- 接口访问日志（可视化界面）
- 异常告警（邮件/钉钉 Webhook）

### 4.2 前端模块（基于原子设计）
#### 4.2.1 原子（Atoms）
- 按钮、输入框、表格单元、图表容器
- 颜色与排版样式（CSS Variables）

#### 4.2.2 分子（Molecules）
- 数据筛选控件（日期选择 + 下拉）
- 表格搜索与分页组件
- 图表工具栏（缩放、图例开关）

#### 4.2.3 组织（Organisms）
- 数据中心面板：历史数据列表 + 图表展示
- 回测结果面板：绩效指标列表 + 可交互图表
- 策略编辑器：代码编辑区 + 保存/运行按钮 + 版本列表

#### 4.2.4 页面（Pages）
- 登录/注册页
- 首页：平台介绍 + 近期回测预览
- 数据浏览页
- 策略开发与管理页
- 回测结果详情页
- 学习资源中心

#### 4.2.5 状态管理与路由
- React Context + Reducer 原子化管理各模块状态
- React Router 实现页面路由
- 数据请求：Axios 实例化配置 + 拦截器

## 五、技术架构
```
[React SPA] ---HTTP/REST---> [FastAPI 服务] ---Akshare SDK---> [数据源]
                                   |               |
                                   |               +--> [本地缓存/Redis]
                                   +--> [数据库(PostgreSQL/SQLite)]
```

## 六、部署与运维
- **后端**：Docker 化部署，Uvicorn+Gunicorn 生产配置
- **前端**：CI 构建并部署到 Nginx 静态站点
- **监控**：Prometheus + Grafana（可选）

## 七、测试计划
- **单元测试**：Pytest、Jest + React Testing Library
- **接口测试**：FastAPI 自带 TestClient
- **端到端测试**：Cypress

## 八、可扩展性与维护
- 插件式策略模块：用户可安装/卸载策略插件
- 多数据源接入：除 Akshare 外，后续可接入 Wind、Tushare 等
- 国际化（i18n）：文本原子化，支持多语言切换

## 九、项目管理与迭代
- 迭代节奏：Sprint 两周
- 版本管理：Git Flow
- 文档与代码规范：Prettier + ESLint + Black + Flake8
