---|---|---

code | 合约代号 | rb1810  
date | 查询日期 | 2018-08-08

参数名 | 类型 | 说明  
---|---|---

shorts | array | 空头龙虎榜  
broker | string | 席位  
long | int | 该席位多头持仓量  
long_chge | int | 该席位多头持仓变化量  
short | int | 该席位空头持仓量
