## 一、数据源扩展

当前仅支持东方财富（非公开 API，有 IP 封禁和频率限制风险），需要多源支持提升可靠性与覆盖面。

### 🔴 多数据源容错架构

- [ ] 抽象 `DataProvider` trait，当前 EastMoney 实现作为第一个 provider
- [ ] 实现数据源自动降级：主源超时/报错 → 自动切换备用源
- [ ] 统一数据格式转换层（不同源的字段名、精度、时区可能不同）
- [ ] 每个数据源独立的频率限制器（当前硬编码 60 req/min）

### 🟡 美股数据源

- [ ] Yahoo Finance API（免费，延迟 15 分钟，覆盖面广）
- [ ] Alpha Vantage（免费 tier 5 req/min，日线数据质量好）
- [ ] Polygon.io（免费 tier 支持日线历史数据）
- [ ] 前端支持美股代码格式（AAPL、MSFT），与 A 股代码（000001）区分
- [ ] 后端路由支持 `market` 参数：`/api/stock/daily?symbol=AAPL&market=us`
- [ ] 前端增加市场切换 UI（A 股 / 美股 tab 或下拉）

### 🟡 港股数据源

- [ ] 东方财富港股接口（与 A 股类似，需要不同的 secid 前缀）
- [ ] 港股代码格式支持（00700.HK）
- [ ] 港股指数快照（恒生指数、恒生科技指数）

### 🟢 加密货币数据

- [ ] Binance 公开 API（免费，无需认证，K 线数据完整）
- [ ] CoinGecko API（免费 tier，支持市值和基本面数据）
- [ ] 24/7 交易时间适配（现有逻辑假设交易日概念）

### 🟢 数据源管理 UI

- [ ] 设置页增加数据源配置面板
- [ ] 支持用户填入自己的 API Key（Alpha Vantage 等）
- [ ] 数据源健康状态展示（在线/离线/限流中）

## 二、策略与回测增强

### 🔴 更多策略

- [ ] KDJ 指标策略（随机指标，短线交易常用）
- [ ] 均线多头排列策略（趋势判断）
- [ ] 成交量突破策略（放量突破前高）
- [ ] 海龟交易法则（经典趋势跟踪）
- [ ] 网格交易策略（震荡市场适用）

### 🟡 策略参数优化

- [ ] 回测页面支持自定义策略参数（当前使用默认值）
- [ ] 参数扫描功能：批量回测不同参数组合，找最优参数
- [ ] 参数优化结果可视化（热力图 / 3D 表面图）

### 🟡 回测引擎增强

- [ ] 支持做空（当前仅支持做多）
- [ ] 滑点模拟（市价单与限价单的成交价差异）
- [ ] 可配置手续费（当前硬编码万三）
- [ ] 印花税模拟（卖出时千一，A 股特有）
- [ ] 多策略组合回测（同时运行多个策略，对比收益）
- [ ] 基准收益线（买入持有 / 沪深300 对比）

### 🟢 策略编辑器

- [ ] 用户自定义策略（简单表达式 DSL 或可视化拖拽）
- [ ] 策略保存与导入导出

## 三、测试与质量保障

### 🔴 CI/CD 流水线

- [ ] GitHub Actions 工作流：
  - `cargo test` — 后端测试
  - `cargo clippy` — Rust lint
  - `cargo fmt --check` — Rust 格式检查
  - `npm run lint` — 前端 ESLint
  - `npm run build` — 前端构建检查
- [ ] PR 合并前强制通过所有 checks
